Project Overview

Application and Purpose

The purpose of this application is to create a professional and promotional website for a local golf shop. The website will provide an online presence for the shop, allowing customers to explore products and access important details. Key features include a direct link to eBay listings, shop location, and contact information.

Intended Users

  • Current Customers
  • Future Customers

Overview of the Content

The website will consist of multiple pages, including:

  • Home Page - Introduction to the golf shop
  • Products Page - Showcase available golf equipment
  • About Us Page - Information about the shop's history
  • Contact Page - Store location, phone number, and email
  • eBay Listings - External link to the shop's online store
